# 🎓 Student Placement Prediction

A Machine Learning web application built using Streamlit that predicts whether a student is likely to be placed based on academic and skill-related information.

## Features

- Predict student placement
- Interactive dashboard
- Modern user interface
- Placement probability
- Machine Learning model

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Dataset

The dataset contains 45,000 student records with the following features:

- Age
- Gender
- Degree
- Branch
- CGPA
- Internships
- Projects
- Coding Skills
- Communication Skills
- Aptitude Test Score
- Soft Skills Rating
- Certifications
- Backlogs

Target:
- Placement Status

## How to Run

1. Clone the repository

2. Install the dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
streamlit run app.py
```
Link for the app:
https://studentplacementprediction.streamlit.app/
